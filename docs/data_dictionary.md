# Data Dictionary

This document summarizes the main variables used in the research workflow.

## Outcome Variable

| Variable | Description | Availability |
|---|---|---|
| depression_prevalence | Treatment-based depression prevalence estimated at the administrative-dong level | Restricted / not shared |

| Category | Variable | Description |
|---|---|---|
| Demographic | female_population_ratio | Share of female population |
| Demographic | older_adult_ratio | Share of older adults |
| Household | single_person_household_ratio | Share of single-person households |
| Socioeconomic | low_income_ratio | Share of low-income population |
| Socioeconomic | low_education_ratio | Share of low-education population |
| Economic / Medical Access | official_land_price | Area-weighted average official land price |
| Economic / Medical Access | mental_health_clinics | Number of psychiatry-related clinics |

Restricted health-insurance data and personally identifiable information are not included in this repository.
